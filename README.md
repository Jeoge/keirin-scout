# 🚴 KEIRIN SCOUT - 競輪予想分析ツール

3連対率・ライン構成・セット消えロジックで買い目を自動生成する競輪予想分析ツールです。

## 機能

- **出走表入力** - 選手データ（3連対率・B/S値・ライン）を入力
- **自動分析** - セット消え・3連対率フィルターで有効選手を絞り込み
- **買い目生成** - 3連単の推奨買い目を最大15点生成
- **穴狙いモード** - 高B/S値選手を軸にした追加買い目を生成
- **Chrome拡張連携** - KEIRIN.jp / Gamboo の出走表を自動取得
- **過去記録** - レース結果・的中チェック・収支管理
- **精度分析** - 的中率・回収率・月別成績を自動集計

## Chrome拡張機能

KEIRIN.jp および Gamboo（my.keirin.kdreams.jp）の出走表データを  
ワンクリックでアプリに自動入力できるChrome拡張機能も配布しています。

## 使い方

1. 出走表を見ながら選手データを入力（または拡張機能で自動取得）
2. ライン構成とポジション（先行/追込）を設定
3. 「分析する」ボタンを押して買い目を確認
4. レース後に「的中チェック」で結果を記録
